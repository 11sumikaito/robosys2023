## plusコマンド
![test](https://github.com/11sumikaito/sumikaito/actions/workflows/test.yml/badge.svg)

## 何をするためのソフトか
plusのプログラムは標準入力から受け取った数値（整数または浮動小数点数）を合計して出力するものです。

## 必要なソフトウェア
 * Python
  * テスト済み: 3.7～3.10

## どんな環境で使えるか
Windows10のUbuntu 20.04 on Windowsで実行可能です。
